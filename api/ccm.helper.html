(static) val(string, regexopt) → {string}

validate a given string

This function returns the given string only if it matches the given regular expression. The result is null if the given string not matches the given regular expression. The result is the return value of ccm.helper.htmlEncode(string) if the given regular expression equals true (string === true). The result is the return value of ccm.helper.noScript(string) if the optional given regular expression is a falsy value (false, null, undefined, 0, NaN, '' or ""). The given regular expression passes ccm.helper.regex(regex) if the given regular expression is a string.
Parameters:
Name Type Attributes Description
string string given string
regex ccm.regex | string | boolean <optional>
 given regular expression
Source:
  • ccm.js, line 2974
Returns:
returns given string or null
Type
string
Examples
ccm.helper.val( 'Hello, world!', /^[A-Z][a-z ,!]*$/ );  // => 'Hello, world!'
ccm.helper.val( 'Hello, world!', /^[A-Z][a-z]*$/ );     // => null
ccm.helper.val( 'Hello, world!<script>alert("XSS");</script>', true );
// => 'Hello, world!<script>alert("XSS");</script>' (no alert call)
ccm.helper.val( 'Hello, world!<script>alert("XSS");</script>' );
// => 'Hello, world!' (no alert call)
